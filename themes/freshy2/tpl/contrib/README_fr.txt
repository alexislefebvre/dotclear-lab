Bonjour,

Vous trouverez ici les templates customisés pour les plugins suivants :
- blogRollPage : fichier blogroll.html
- categoriesMode : fichier categories.html
- contactMe : fichier contact_me.html
- contact : fichier contact.idblog.html où idblog est à remplacer par le nom du répertoire de votre blog
- subscribetocomments : fichier subscribetocomments.html

Si vous utilisez un de ces plugins, il faut alors déplacer le fichier .html dans le dossier tpl du thème 
pour qu'il soit pris en compte par le thème Freshy2 qui a une structure peu orthodoxe...

Les thèmes gallery sont placés directement dans leurs dossiers respectifs :
- gallery (version 0.2.16) : fichiers galleries.html, gallery.html et image.html dans le dossier default
- gallery (version 0.3.1) :  fichiers galleries.html, gallery.html et image.html dans le dossier gal_default

En attendant qu'on trouve un moyen pour rendre la structure de Freshy2 plus compatible avec les autres plugins,
si vous avez une demande particulière, merci de me contacter à contact@brol.info et j'essaierai de 
satisfaire votre demande.

Merci ;-)
