Ce plugin permet d'afficher dans la sidebar le nom et la couverture du/des livres qu'on lit et de la musique qu'on écoute régulièrement, du film que l'on est allé voir récemment ou tout autre lien avec image. Deux widgets (liste des catégories Cinecturlink et liste de mes Cinecturlink) et une page publique dédiée aux cinecturlinks sont mis à disposition.

Configuration

Général
Largeur maximale des images (en pixels) :
cette valeur définit la taille de l'affichage de l'image en page publique dédiée aux cinecturlinks, ainsi que dans le widget.
Il est possible de laisser vide ce champ, la taille de l'image sera alors réglée via la feuille de style du thème.
Les valeurs acceptées vont de 0 à 9999.
Par défaut, la valeur est 100 (sans mention d'unité de mesure).
Dossier public des images (dans le répertoire "public" du blog) :
suite à l'enregistrement de la page de configuration, le dossier est automatiquement créé dans le répertoire "public" du blog.
Page publique
Activer la page publique :
cocher la case permet de faire apparaître le lien direct vers la page publique dédiée aux cinecturlinks.
Limiter le nombre d'entrées par page sur la page publique à :
les valeurs acceptées vont de 1 à 9999999999.
Par défaut, la valeur est 10.
Informations
  • Une fois le plugin configuré et vos liens créés, vous pouvez placer un des widgets cinecturlink depuis la page des widgets de présentation.
  • Afin d'ouvrir les liens dans une nouvelle page vous pouvez utiliser le plugin External Links.
  • Afin de changer l'URL de la page publique vous pouvez utiliser le plugin My URL handlers.
  • Vous pouvez ajouter la page publique de cinecturlink au plugin Sitemaps.
  • Le plugin Cinecturlink2 est compatible avec les plugins Rate it et Activity report.
