This plugin will display the name in the sidebar and coverage / books we read, and music we listen to regularly, the film that we recently went to see, or any other related image. Two widgets (Cinecturlink list of categories and list of my Cinecturlink) and a page dedicated to public cinecturlinks are available.

Configuration

General
Maximum width of images (in pixel):
this value defines the size of the display of public image page dedicated to cinecturlinks and in the widget.
You can leave this field empty, the image size will be set via the theme style sheet.
Valid values range from 0 to 9999.
The default value is 100 (without reference to unit of measurement).
Public folder of images (under public folder of blog):
following the registration of the configuration page, the file is automatically created in the "public" directory blog.
Public page
Enable public page:
check the box used to display the direct link to the page dedicated to the public cinecturlinks.
Limit number of entries per page on pulic page to:
accepted values range from 1 to 9999999999.
The default value is 10.
Informations
  • Once the plugin has been configured and your links have been created, you can place one of the cinecturlink widgets in the sidebar.
  • In order to open links in new window you can use plugin External Links.
  • In order to change URL of public page you can use plugin My URL handlers.
  • You can add public pages of cinecturlink to the plugin Sitemaps.
  • The plugin Cinecturlink2 is compatible with plugins Rate it and Activity report.
