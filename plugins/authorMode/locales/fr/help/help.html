Activation

Pour obtenir le widget de présentation ainsi que les pages .html associées, il faut bien évidemment activer le plugin.

Options avancées

Préfixes des URLs

Vous pouvez indiquer le préfixe qui sera utilisé pour accéder à la page publique de l'auteur (par défaut, l'URL author aura comme préfixe author) et pour la page publique regroupant tous les auteurs (par défaut, l'URL authors aura comme préfixe authors).

Options de liste

Lister seulement les auteurs de billets standards : quand cette option est cochée, seuls les billets sont pris en compte (pas les pages, pages connexes, billets muppet ou autres).

Le tri alphabétique n'intervient que sur la liste des auteurs (widget et page regroupant tous les auteurs). Il s'effectue par ordre de préférence (si vide, on passe au suivant) : Pseudonyme -> Prénom -> Nom.

Widget

Le widget affiche, par défaut, le lien vers la liste des auteurs. Si vous voulez ne pas l'afficher, ajoutez à la fin de votre fichier css de thème le code suivant : #sidebar .listauthors {display:none;}
