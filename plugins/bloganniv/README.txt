Bloganniv Widget for Dotclear 2
===============================================================================

Cette version de plugin sert à afficher le nombre de jours restant avant l'anniversaire de votre blog

Il affiche le nombre de jours restant avant l'anniversaire et le nombre d'années de votre blog.

Installation
------------

Placez le dossier "bloganniv" de cette archive dans le dossier "plugins" du répertoire de Dotclear 2.
Rendez vous dans l'interface d'administration où vous y trouverez le nouveau widget et placez-le où bon vous semble :)
