miniSEO for Dotclear 2
===============================================================================

Cette version de plugin sert a ajouter dans la page la donnée meta description,
si le plugin Mymeta n'est pas en place vous les 250 premiers caractères du billet 
servirons alors à remplir le meta description.

Historique
----------
0.3 Couplage avec le plugin http://plugins.dotaddict.org/dc2/details/Mymeta
0.2 Nous ne sommes plus obligé de placer tpl_use_cache sur non
0.1 Première version
Installation
------------

Placez dans votre thème le marqueur {{tpl:MiniSEO}} souvent il suffit d'ajouter cela dans le fichier _head.html


Evolution future
----------------
