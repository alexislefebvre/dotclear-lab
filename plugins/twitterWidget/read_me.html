Twitter Widget

Ce plugin est entièrement basé sur le script de Remy Sharp.

Pour plus d'informations, le contacter sur son blog, ou me contacter sur le mien.

Informations générales

Ce plugin affiche les derniers tweets d'un compte twitter. Grâce au javascript, la page se charge même si twitter rame.

Par défaut, cela crée un code de la forme:

<div class="twitter">
  <h2>Twitter</h2>
  <div id="remy_tweets">
    <ul>
      <li>
        <span class="twitterPrefix">Remy said: </span>
        <span class="twitterStatus">I just had a bizarre spaces moment -
          moving my cursor to the bottom of the screen switched space -
          annoying if it hadn't just gone away.</span>
        <span class="twitterTime">20 minutes ago</span>
      </li>
      <li>
        <span class="twitterPrefix">Remy said: </span>
        <span class="twitterStatus">Great quote - Michael J Fox:
          "my happiness grows in direct proportion to my acceptance,
          and in inverse proportion to my expectations."</span>
        <span class="twitterTime">7 days ago</span>
      </li>
    </ul>
  </div>
</div>

et ça affiche:

Twitter

  • Remy said: I just had a bizarre spaces moment - moving my cursor to the bottom of the screen switched space - annoying if it hadn't just gone away. 20 minutes ago
  • Remy said: Great quote - Michael J Fox: "my happiness grows in direct proportion to my acceptance, and in inverse proportion to my expectations." 7 days ago

Personnalisation du style

On peut ainsi le styliser:

.twitter h2{
  color: #384868;
}
.twitter ul{
  padding: 0;
  list-style: none;
}
.twitter li{
  border-bottom: 1px dotted #384868;
  margin-bottom: 1em;
}
.twitter .twitterPrefix{
  color: #384868;
  text-tranform: uppercase;
}
.twitter .twitterStatus{
  font-style: italic;
}
.twitter .twitterTime{
  font-weight: bold;
}

Template HTML

Le champ template peut supporter plusieurs variables:

  • %text% - Le message.
  • %id% - L'id du message.
  • %source%
  • %created_at%
  • %time% - Depuis combien de temps le message a été posté.
  • %user_name% - Le vrai nom de l'utilisateur.
  • %user_screen_name% - Le nom d'utilisateur Twitter.
  • %user_id%
  • %user_profile_image_url% - L'avatar de l'utilisateur.
  • %user_url% Le site de l'utilisateur.
  • %user_location%
  • %user_description%

Par exemple, ce template:

<span class="twitterPrefix">
<img height="16" width="16" src="%user_profile_image_url%" />
<a href="http://twitter.com/%user_screen_name%">%user_name%</a> said: </span>
<span class="twitterStatus">"%text%"</span>
<span class="twitterTime"><a href="http://twitter.com/%user_screen_name%/statuses/%id%">%time%</a></span>

affichera (pour le compte Twitter de Dotclear):
