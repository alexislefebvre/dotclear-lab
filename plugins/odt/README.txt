Plugin export ODT pour Dotclear
===============================

Toutes les infos :
http://aurelien.bompard.org/post/2009/06/05/Export-des-billets-Dotclear-en-ODT

Dépendances :
 * Le module XSL de PHP, disponible dans toutes les bonnes distribs, dont le
   nom du paquet est en général “php-xsl”
 * PHP >= 5.2 pour la bibliothèque odtPHP que j’utilise

Installation :
Installez le plugin, et ajouter le nouveau widget "Export ODT". Et voilà !

Have fun !

Aurélien
