Ce plugin permet aux visiteur d'exporter une page du blog (ou la page d'accueil du blog) au format OpenDocument (ODT).

Cette page vous permet de contrôler differents aspects du plugin. Pour afficher le lien d'export ODT, un "widget de présentation" est également disponible.

Bouton d'export
Ce champ permet de placer le bouton d'export sur la page du billet.
Importer les images distantes
Pendant la génération du document ODT, Dotclear peut télécharger les images distantes que vous avez incluses dans votre billet. L'avantage est que les images seront intégrées dans le document ODT, le rendant plus rapide à charger et plus "indépendant". L'inconvénient est que la génération peut être plus longue, voire même échouer si le serveur hébergeant votre blog n'est pas autorisé à télécharger depuis Internet.

Si une erreur de configuration est affichée, cela signifie que votre installation de PHP ne dispose pas du module XSLT. Si votre blog est hébergé sur une distribution Linux, le module nécessaire se trouve dans un paquet probablement nommé "php-xsl".
