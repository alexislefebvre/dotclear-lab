À propos

Cette extension permet d'ajouter simplement une licence à tous le fichier d'une extension.
Les changements se font sur les modules installés. Ces changements sont irréversibles.

Paramètres des fichiers

Écraser la licence existante
Réécrit la licence dans les fichiers même si elle existe déjà. (recommandé)
Ajouter le fichier de licence complet à la racine du module
Comme son nom l'indique, cela ajoute la licence complête dans le fichier LICENSE à la racine du module. (recommandé)
Ajouter le bloc de licence aux ficheirs PHP
Permet d'ajouter le bloc de licence court aux fichiers php du module. (recommandé)
Ajouter le bloc de licence aux ficheirs JS
Permet d'ajouter le bloc de licence court aux fichiers javascript du module.
Ne pas ajouter la licence aux fichiers du dossier locales
Les fichiers de traduction présent dans le réperoire locales du module ne seront pas changés. A noter que si un répertoire libs est présent dans le module, celui ci ne sera pas changé non plus.

Paramètres des licences

Les entêtes sont les résumés de licence. Elles sont placées en haut de chaque fichier d'un module sous forme de commentaire. Pour les plugins il est préférable de choisir la licence GPL2 qui correspond à la licence de Dotclear. Il est possible de modifier une entête et certaines informations directement en rapport avec le module peuvent être écrites dans une entête en utilisant les masques suivants :

%year%
Année en cours.
%module_id%
Id du module (nom correspondant au dossier du module).
%module_name%
Le nom du module tel qu'il est défini dans le fichier _define.php.
%module_author%
Le nom de l'auteur du module tel qu'il est définie dans le fichier _define.php.
%module_type%
Soit plugin, soit thème.
%user_cn%
Le nom commun de l'utilisateur qui utilise licenseBootstrap.
%user_name%
Le nom de l'utilisateur qui utilise licenseBootstrap.
%user_email%
L'adresse email de l'utilisateur qui utilise licenseBootstrap.
%user_url%
L'adresse web de l'utilisateur qui utilise licenseBootstrap.

Pour chaque licence présente dans cette liste, un fichier LICENSE complet est également disponible.
