Comment insérer Plan du site dans l'interface du blog ?

Avec un widget :
Le plus simple est d'utiliser le widget "Plan du site" qui propose un lien direct vers la page dédiée.

Dans le fichier template :
Le lien vers la page "Plan du site" peut aussi être intégrée dans l'interface du blog en éditant directement votre fichier template. Il suffit pour cela d'ajouter l'instruction <a href="{{tpl:SiteMapURL}}">{{tpl:lang Sitemap}}</a> à l'endroit voulu.
