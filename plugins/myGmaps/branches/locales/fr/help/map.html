Types d'élements de cartes

Selon les clics sur la carte, un ou plusieurs marqueurs Gmap seront insérés et la carte sera rafraîchie. Les marqueurs peuvent être déplacés. Un élément de carte peut avoir les types suivants :

Type non défini
Aucun marqueur n'est présent sur la carte, cet élément ne peut être utilisé.
Point d'intérêt
Un seul marqueur est présent sur la carte. Ce point sera affiché avec comme balise title le titre de l'élément de carte. Un clic sur le point affichera le contenu de l'élément dans l'infobulle.
Parcours
Plusieurs marqueurs sont présents sur la carte, reliés par une ligne. Cette ligne sera affichée sur la carte. Un clic sur la ligne affichera le contenu de l'élément dans l'infobulle.
Fichier kml inclus
Un fichier kml peut-être inclus dans la carte. Il n'est pas modifiable.

Rédaction de l'élément de carte

Titre de l'élément
Inscrivez le titre de l'élément de carte. Ce champ est obligatoire.
Coordonnées
Cliquez sur la carte, un ou plusieurs marqueurs Gmap seront insérés et la carte sera rafraîchie. Les marqueurs peuvent être déplacés. Si un seul marqueur est affiché, un clic permet de choisir son icône. Si plusieurs marqueurs sont affichés, le clic permet de choisir la couleur du parcours.
Description
La description du point d'intérêt ou du pacours contenus dans la carte. Cette description s'affichera sur le blog dans une infobulle lors du clic sur l'icône du point d'intérêt ou sur la ligne du parcours. Si ce champ est laissé vide, seul le titre s'affichera dans l'infobulle.
Notes
Cette zone de texte sert à la prise de notes ou pense-bête divers. Elle ne sera jamais affichée.
Catégorie
La catégorie de votre élément de carte. Pour créer une nouvelle catégorie, rendez vous dans la section "catégories". Vous pouvez n'affecter votre élément de carte à aucune catégorie en choisissant la ligne vide.
État de l'élément de carte
Permet de choisir l'état de l'élément de carte après enregistrement :
  • en attente : l'élément de carte est en attente de publication. Il ne peut pas être inséré dans un billet
  • publié : l'élément de carte est disponible pour l'ajout dans un billet.
Publiée le
Permet de modifier la date et l'heure de publication de l'élément de carte.
Format du texte
Permet de choisir la syntaxe de saisie utilisée pour la description de l'élément de carte. Le wiki est une syntaxe simplifiée et sera converti en xhtml valide ; à moins que vous maîtrisiez parfaitement le HTML nous vous conseillons de la choisir de préférence. Consultez la référence de la syntaxe Wiki pour plus d'informations.
