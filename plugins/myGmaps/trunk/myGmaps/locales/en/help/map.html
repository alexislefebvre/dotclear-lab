Editing a map element

Map element title
Type in the map element title. This field is mandatory.
Coordinates
Clic on the map, one or several markers will be inserted and the map will be refreshed. Markers can be dragged on the map. If only one marker is displayed, one clic on it will allow to choose its icon. If several markers are displayed, one clic on any of them will allow to choose the polyline color.
Description
Enter here a description for the point of interest or polyline included in the map element. This description will be displayed on the blog in an infowindow when users click on the markers or the polyline. If this field is left empty, only the title will be displayed in the infowindow.
Notes
This text area is for personal use, for notes or memos. What you enter here will never be displayed on the blog..
Category
Your map element's category. To create a new category, please see the "categories" section. You can choose to leave your map element without a category by selecting the blank line.
Map element status
Allows you to choose your map element's status:
  • pending: the map element is pending completion, it is not available for use in a post.
  • published: the map element is ready to be included in a post.
Published on
Here you can change the map element's publication date and time.
Text formating
To choose the map element editing syntax. Wiki is a simplified syntax and will be converted to valid xhtml ; Unless you have a good understanding of html, we advise you to choose the wiki syntax. See the Wiki syntax reference for more information.
