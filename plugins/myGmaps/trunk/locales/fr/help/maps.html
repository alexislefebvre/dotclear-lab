Présentation

Insérer des cartes Gmap dans vos billets.
Ce plugin permet de créer et éditer des éléments de cartes. Ces éléments seront utilisés pour insérer des cartes Gmap dans vos billets.

Paramètres

Options des cartes par défaut
Déplacer le pointeur sur la carte pour choisir le centre par défaut. Si nécessaire, vous pouvez utiliser le zoom qui sera aussi mémorisé.

Filtres de la liste des éléments

Auteur
Permet de filtrer les éléments de cartes par auteur.
Catégorie
Filtrer les éléments de cartes par catégorie.
État
  • en attente : en attente de publication.
  • publié : élément de carte en ligne.
Mois
Filtre les éléments de cartes d'un mois d'une année donné.
Type d'élément de carte
Filtre les éléments de cartes selon le type indiqué.
Trier par
Permet de trier les résultats de filtrage selon la date, le titre, la catégorie, l'auteur ou l'état de publication.
Trier
Indique l'ordre dans lequel on souhaite effectuer le tri.
Éléments de cartes par page
Nombre d'éléments de cartes à afficher par page de résultat.

Actions par lot sur les éléments

Il est possible d'effectuer un ensemble d'actions sur plusieurs éléments de cartes, d'un seul coup. Les actions possibles dépendent des permissions de l'utilisateur.

  • Publier : mettre l'élément de carte en ligne,
  • Mettre en attente : l'élément de carte est en attente de publication,
  • Changer de catégorie : envoie sur la liste des catégories pour changer celle des éléments de cartes sélectionnés,
  • Changer l'auteur : permet de changer l'auteur de l'élément de carte en indiquant l'identifiant de l'utilisateur qui deviendra le nouvel auteur,
  • Supprimer : supprime l'élément de carte (cette opération est irréversible).
