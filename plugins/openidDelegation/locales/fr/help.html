OpenID est un standard ouvert qui permet de s'authentifier sur des sites web sans avoir à créer un nouveau mot de passe. La délégation OpenID permet d'utiliser l'adresse de votre blog comme identifiant OpenID, en déléguant les demandes à un autre fournisseur.

Vous pouvez ainsi vous créer un compte sur un fournisseur OpenID gratuit, par exemple myOpenID ou OpenID France, et lui déléguer l'autentification. Ainsi, si vous désirez changer de fournisseur, votre identifiant OpenID restera l'adresse de votre blog.

Cette délégation est réalisée par l'ajout d'instructions dans l'en-tête de votre page d'accueil, qui peuvent être générées par ce plugin.

Utiliser la délégation OpenID
Permet d'activer ou de désactiver globalement les en-têtes permettant la délégation OpenID
Fournisseur
Choisissez l'un des fournisseurs OpenID pré-enregistrés. Si votre fournisseur n'est pas dans la liste, vous pouvez choisir "Personnalisé" et copier-coller directement les instructions de délégation.
Nom d'utilisateur
Le nom de votre compte chez votre fournisseur OpenID, s'il est pré-enregistré.
