À propos de l'aide

Cette page d'aide ne contient pas d'informations sur l'utilisation d'un compte PayPal. Reportez-vous à l'abondante documentation du service en cas de doute sur sa configuration.

Configuration

Activer l'extension

Si vous cochez cette case, l'extension sera activée sur ce blog.

Utiliser la Sandbox PayPal pour les transactions

Si vous cochez cette case, les transactions seront effectuées sur votre compte fictif de test dans la Sandbox Paypal. Utilisez cette option pour faire vos tests.

Identifiant du compte de test

Entrez dans ce champ l'adresse email associée à votre compte marchand dans la Sanbox PayPal. Les paiements seront effectués sur ce compte si vous avez activé l'utilisation de la Sandbox PayPal.

Identifiant du compte marchand

Entrez dans ce champ l'adresse email associée à votre compte PayPal ou votre ID marchand sécurisé. Les paiements seront effectués sur ce compte si vous avez désactivé l'utilisation de la Sandbox PayPal.

Monnaie

Choisissez la monnaie dans laquelle les paiements seront effectués sur votre compte.

Options avancées

Activer le cryptage SSL des boutons

Si vous cochez cette case, les boutons seront cryptés. Toutefois, vous ne pourrez utiliser cette option que si vous pouvez accéder à votre serveur en SSH et générer vos certificats.

  1. rendez-vous dans le répertoire /public/paypalbuttons/ et entrez la commande suivante pour générer votre clé privée : 
    openssl genrsa -out clepriv.pem 1024
  2. entrez la commande suivante pour générer votre certificat public : 
    openssl req -new -key clepriv.pem -x509 -days 365 -out certpub.pem
    Remplissez les informations demandées.
  3. téléchargez le certificat public de Paypal en vous rendant sur la page Mon Compte => Préférences => Plus d’Options => Certificats de cryptage pour site marchand. Renommer le fichier téléchargé en paypal_cert.pem et placez-le dans le répertoire /public/paypalbuttons/ qui doit maintenant contenir 3 fichiers :
    • certpub.pem
    • clepriv.pem
    • paypal_cert.pem

Chemin vers OpenSSL sur votre serveur

Entrez ici le chemin, depuis la racine du serveur, vers l'emplacement du programme OpenSSL

ID de votre certificat PayPal

Dans votre compte PayPal, sur la page des certificats, dans la partie « Vos certificats publics », cliquez sur le bouton « Ajouter » et sélectionnez le fichier certpub.pem que vous avez précédemment généré. Vous reviendrez ensuite sur la page de gestion des certificats et vous devriez voir apparaitre une ligne dans le tableau du bas avec comme informations l’ID du certificat, l’autorité de certification (correspondant aux informations saisies lors de la génération de votre certificat public), et la date d’expiration de celui-ci. Copiez l’ID du certificat et collez-le dans le champ.
