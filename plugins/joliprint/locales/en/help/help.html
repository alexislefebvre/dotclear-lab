C'est quoi?

Le bouton Joliprint permet au visiteur d'imprimer un article au format PDF.

Widget

Un widget est disponible. Il s'affichera sur les pages reconnues comme étant des articles.
L'extension doit être activée dans la page d'administration du plugin.

Contenu d'un article

Un bouton peut être placé automatiquement avant et/ou après le contenu d'un article.
Pour cela le thème utilisé doit avoir les behaviors publicEntryBeforeContent et publicEntryAfterContent. (voir la documentation Dotclear)
La configuration de l'aspect du bouton et de son emplacement se fait depuis la page d'administration du plugin.
L'extension doit être activée dans la page d'administration du plugin.

Balise de template

Une balise de template nommée 'joliprintButton' est disponible. Elle peut être placée n'importe ou dans un thème. Elle sera active si elle est placée dans le contexte d'un article.
Les attributs 'url', 'server', 'button' et 'text' sont disponibles mais facultatif.

url
Lien vers la page à imprimer.
server
Zone la plus proche de votre blog.
Soit eu.joliprint.com pour l'Europe,
soit api.joliprint.com pour le reste.
button
Style d'image du bouton.
Button : joliprint-button.png,
Button big : joliprint-button-big.png,
Icon : joliprint-icon.png,
Icon small : joliprint-icon-small.png,
Icon pdf : pdf-icone.gif
text
Texte à afficher à coté du bouton.

Exemple:
{{tpl:joliprintButton button="pdf-icone.gif" text="Sauvegarder en PDF"}}

L'extension doit être activée dans la page d'administration du plugin.
