Configuration

Depuis la page "Paramètres du blog", vous pouvez activer le bouton Acronyme sur la barre d'outils de rédaction de billet.
Vous pouvez également activer la page publique des acronymes utilisés. Le widget "Liste des acronymes" sera alors disponible dans le menu "Widgets de présentation".

Utilisation

Les acronymes de cette liste seront automatiquement reconnus par le système lorsque vous utiliserez la syntaxe wiki. C'est-à-dire que vous n'aurez pas à saisir le titre, simplement encadrer l'acronyme de doubles points d'interrogation.

Exemple : ??ANSI??

Seuls les acronymes ajoutés dans cette liste se retrouveront dans la page publique des acronymes utilisés.
Les acronymes ajoutés grâce au bouton Acronyme de la barre d'outils de rédaction de billet ne sont pas automatiquement ajoutés à cette liste.

Note : pour supprimer un acronyme, vider simplement son libellé.
