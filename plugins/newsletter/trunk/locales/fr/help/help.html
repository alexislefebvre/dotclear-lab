Cette extension permet de gérer l’envoi d’une newsletter.

Onglet Abonnés

Il est possible d’effectuer un ensemble d’actions simultanément sur plusieurs abonnés.

Modifier l’état :
  • Suspendre : suspend les comptes
  • Désactiver : supprime les comptes
  • Activer : active les comptes
  • Dernier envoi : réinitialise la date du dernier envoi à la date d’inscription de l’abonné
Courriel à envoyer :
  • Lettre d’informations : envoie la newsletter
  • Confirmation : envoie la confirmation de création du compte
  • Suspension : envoie le mail de suspension du compte
  • Désactivation : envoie le mail de suppression du compte
  • Activation : envoie le mail pour activer le compte
Modifier le format :
  • html : change le format d’envoi en html pour les comptes sélectionnés
  • texte : change le format d’envoi en texte pour les comptes sélectionnés

Onglet Ajouter

Ajouter un abonné :
  • Email : ajoute l’email défini à la liste des abonnés

Onglet Paramètres

Paramètres :
  • Nom de l’éditeur : définit le nom de la personne qui emet la newsletter
  • Email de l’éditeur : définit le courriel de la personne qui emet la newsletter
  • Message de présentation : définit le message de présentation de la newsletter
  • Message de présentation pour les billets : définit le message de présentation de la liste des billets
  • Introduction au lien de confirmation : définit le message de présentation avant le lien de confirmation
  • Titre du lien de confirmation : définit le titre du lien, par défaut (cliquez ici)
  • Sujet du mail de confirmation : définit le sujet du mail
  • Introduction au lien de désactivation : définit le message de présentation avant le lien de désactivation
  • Titre du lien de désactivation : définit le titre du lien, par défaut (cliquez ici)
  • Sujet du mail de désactivation : définit le sujet du mail
  • Introduction au lien d’activation : définit le message de présentation avant le lien d’activation
  • Titre du lien d’activation : définit le titre du lien, par défaut (cliquez ici)
  • Sujet du mail d’activation : définit le sujet du mail
  • Introduction au lien de suspension : définit le message de présentation avant le lien de suspension
  • Titre du lien de suspension : définit le titre du lien, par défaut (cliquez ici)
  • Sujet du mail de suspension : définit le sujet du mail
  • Sujet du mail de résumé : définit le sujet du mail
  • Sujet du mail de changement de mode : définit le sujet du mail
  • Sujet de la Newsletter : définit le sujet du mail
  • Message d’introduction : définit un message personnel en début de newsletter
  • Message de conclusion : définit le message de politesse
  • Message de présentation du formulaire : définit un message d’information avant le formulaire
Paramètres avancés :
  • Code visuel : ajout d’un code visuel de vérification pour éviter les robots
  • Format par défaut pour l’envoi : définit le format d’envoi par défaut
  • Utiliser le format par défaut pour l’envoi : définit si on utilise le format d’envoi par défaut ou par utilisateur
  • Envoi automatique : Cocher cette case pour activer l’envoi de la newsletter suite à la publication d’un billet
  • Accusé de réception : Cocher cette case pour recevoir un accusé de réception à chaque envoi de mail pour chaque destinataire
  • Nombre minimal de billets : définir le nombre minimal de billets à afficher dans la newsletter
  • Nombre maximal de billets : définir le nombre maximum de billets à afficher dans la newsletter
  • Affiche le contenu des billets : affiche le contenu des billets dans la newsletter
  • Taille maximale du contenu des billets : définir la taille du contenu des billets dans la newsletter
  • Catégorie : filtre les billets à envoyer dans la catégorie sélectionnée
  • Utiliser l’option suspendre : Cocher cette case pour utiliser l’option de suspension de compte

Onglet Planification

Cet onglet permet de définir une tâche planifiée pour l’envoi de la newsletter en utilisant l’extension nommée dcCron pour Dotclear 2.

Planification de la newsletter :
  • Intervalle de temps en secondes entre 2 exécutions : définit le temps entre chaque envoi
  • Date de la première exécution : spécifie la date du premier envoi
  • Planifier : crée la tâche planifiée NewsletterPlan avec les paramètres précédents
  • Déplanifier : supprime la tâche planifiée NewsletterPlan
Tâche planifiée : NewsletterPlan
Affiche les caractéristiques de la tâche planifiée.
  • Etat : affiche l’état actuel de la tâche planifiée
  • Intervalle : affiche l’intervalle actuellement défini
  • Dernière exécution : affiche la date de la dernière exécution
  • Prochaine exécution : affiche la date de la prochaine exécution
  • Temps restant : affiche le temps restant avant la prochaine exécution
  • Activer : active la tâche planifiée NewsletterPlan
  • Désactiver : désactive la tâche planifiée NewsletterPlan

Onglet Maintenance

Etat de l’extension :
Cocher cette case pour activer l’extension.
Import/Export des données :
Choisir le mode pour l’export de la liste des abonnés.
  • Ce blog seulement : crée le fichier public/nomblog-newsletter.dat
  • Toutes les données : crée le fichier public/newsletter.dat
Adapter le template pour le thème :
Créer une adaptation du template générique pour s’adapter correctement au thème.
  • Nom du thème : sélectionner le thème désiré (ne sont disponibles que les thèmes avec une adapation du home.html et du post.html)
  • Adapter : lancer la création du fichier template subscribe.newsletter.html dans le tpl du thème
  • Cliquer ici pour tester le template : test de l’affichage du template subscribe.newsletter.html
Suppression de toutes les informations concernant newsletter dans votre base de données :
Suppression : cliquer sur ce bouton pour désactiver l’extension, supprimer la table (prefix)_newsletter, supprimer les valeurs du paramétrage dans settings et supprimer la version.

Configuration du widget

Mise en place du widget :
Déplacer le widget "Lettre d’informations" vers l’un des bandeaux.
Paramétrage du wigdet :
  • Titre : définition du titre
  • Afficher le titre : cocher cette case pour afficher le titre du widget
  • Page d’accueil uniquement : cocher cette case pour afficher sur la page d’accueil uniquement
  • Dans un widget : cocher cette case pour afficher le formulaire d’abonnement directement sur la page d’accueil
  • Dans un lien : cocher cette case pour afficher un lien vers la page du formulaire d’abonnement
  • Titre du lien de souscription : définition du lien vers le formulaire d’abonnement

Thème

Intégration au thème :
  • Vous pouvez ajouter le code suivant à votre thème pour afficher le formulaire d’abonnement à la newsletter : 
    {{tpl:NewsletterSubscription}}
    NB : fonction non testée en 3.5.x
