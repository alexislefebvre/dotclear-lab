Description

prvcat vous permet de définir tout une catégorie comme privée. Lorsque vous définissez une catégorie comme privée, le mot de passe que vous avez déterminé est attribué à tous les billets de cette catégorie. Vous pouvez ensuite consulter les billets en utilisant le lien direct, après avoir rentré le bon mot de passe. Les billets sont également listés sur la page de la catégorie qui n'est accessible qu'après avoir entré le bon mot de passe.

Différences par rapport à la gestion de base

prvcat permet une gestion globale des billets privés : un seul mot de passe pour tous les billets. De plus, l'accès aux billets privés est plus simple : il suffit de se rendre sur la page de la catégorie. D'un autre côté, le fait d'avoir les catégories privées listées, mais inaccessibles sans mot de passe, permet aux visiteurs de se rendre compte facilement qu'il existe des billets privés sur votre blog. Si vous ne souhaitez pas qu'il découvre l'existence même de billets privés, n'utilisez pas ce plugin.

Pour les catégories qui ne sont pas gérées par prvcat, le fonctionnement des billets privés n'est pas altéré. Vous pouvez avoir une ou plusieurs catégories privées, ainsi que des billets privés « classiques » dans d'autres catégories.
