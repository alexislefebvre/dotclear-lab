Cette extension permet d'afficher la liste des commentaires et rétroliens dans la partie publique de votre blog.

Configuration de l'extension

Activation de l'extension :
Cocher cette case pour activer l'extension.
Options générales :
  • Titre de la page : définition du titre
  • Nombre par page : choisissez le nombre de commentaires par page
  • Sens de tri : choisissez l'ordre de tri pour l'affichage de la liste des commentaires

Configuration du widget

Mise en place du widget :
Déplacer le widget disponible vers l'un des bandeaux.
Paramétrage du wigdet :
  • Titre : définition du titre
  • Titre du lien : nom du lien
  • Page d'accueil uniquement : cocher cette case pour afficher sur la page d'accueil uniquement
