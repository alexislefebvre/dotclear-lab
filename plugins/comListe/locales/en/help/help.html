The plugin allows to printing the list of comments and trackbacks in public part of your blog.

Configure plugin

Enable plugin :
Check this box to enable the plugin.
Options :
  • Title page : choose title page
  • Comments per page : Allows you to choose the number of comments on each page of this list.
  • Sort : Choose whether the list will be sorted in ascending or descending order.

Configure widget

Enable widget :
Drag and drop the plugin.
Options :
  • Title : choose title widget
  • Link name : choose the name of the link
  • Homepage only : check this box for homepage only
