That plugin add an amazing tag cloud using a 3D animation flash.

You can choose animation size with units using pixel (px), points, (pt) or percentage (%).

You can choose colors :

  • background color
  • two colors for tags
  • a color for mouseover tag

You can also choose define rotation speed for tags. High number for small speed and small number for high speed. Default speed is 100.

Usage

Enable the plugin (tab Settings)

Define params (size, colors, ...)

Installation

Go to widget page et install "Tags flash" widget.

You can change title or let it blank.

You can add list of tags to be seo friendly. It add tags in a classic ul/li list.

You can use transpency mode but background will have no effect.

And you can also limit the number of tags.
