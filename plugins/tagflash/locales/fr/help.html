Ce plugin permet de présenter le nuage de tags sous la forme d'une animation flash. Les tags sont répartis sur une sphère en 3D.

Vous pouvez choisir les dimensions de l'animation, en précisant les unités : pixels (px), points (pt) ou pourcentage (%).

Vous pouvez aussi définir les couleurs :

  • La couleur de fond
  • Deux couleurs pour les tags
  • Une couleur pour le tag survolé par la souris

Vous pouvez aussi choisir la vitesse de rotation des tags. Plus la vitesse est proche de zéro plus les tags tournent rapidement. La valeur par défaut est 100.

Usage

Activer le plugin dans l'onglet paramètre.

Définissez les différents paramètres (dimensions, couleurs,...)

Mise en place

Allez sur la page "widget de présentation" et mettez en place le widget "Tags Flash"

Vous pouvez changer le titre ou ne pas en mettre

Vous pouvez ajouter les tags classiques (liste ul/li) comme contenus alternatifs pour les moteurs de recherche

Vous pouvez choisir le mode transparent mais dans ce cas la couleur de fond choisie n'aura aucun effet.

Et finalement vous limiter le nombre de tags affichés.
