Enregistrer
Permet d'enregistrer une modification/création d'un(e) tâche/objectif
Undo
Permet d'annuler la modification en cours
Crayon et papier
Permet de modifier l'intitulé et la description d'un(e) tâche/objectif.
Crayon +
Permet d'ajouter une tâche associée à l'objectif qui est sur la même ligne que l'icone.
Crayon -
Permet d'annuler l'ajout d'une tâche associée à l'objectif qui est sur la même ligne que l'icone.
Case à cocher
Pour les objectifs: Permet d'afficher ou non l'objectif publiquement sur votre blog.
Pour les tâches: Permet d'indiquer si la tâche a été effectuée
Poubelle
Permet de supprimer un(e) tâche/objectif, définitivement. Une popup de comfirmation permet de prévenir les risques
