Enregistrer
Permet d'enregistrer une modification/création d'un(e) tâche/objectif.
Undo
Permet d'annuler la modification en cours.
Crayon et papier
Permet de modifier l'intitulé et la description d'un(e) tâche/objectif.
Crayon +
Permet d'ajouter une tâche associée à l'objectif qui est sur la même ligne que l'icône.
Crayon -
Permet d'annuler l'ajout d'une tâche associée à l'objectif qui est sur la même ligne que l'icône.
Case à cocher
Pour les objectifs : permet d'afficher ou non l'objectif publiquement sur votre blog.
Pour les tâches : permet d'indiquer si la tâche a été effectuée.
Poubelle
Permet de supprimer un(e) tâche/objectif, définitivement. Une popup de confirmation permet de prévenir des risques
