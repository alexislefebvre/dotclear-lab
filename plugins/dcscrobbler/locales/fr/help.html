Service et application Last.fm

Le widget dcScrobbler vous permet d'afficher sur votre blog les morceaux que vous écoutez avec votre lecteur habituel.

Pour cela il utilise les flux générés par le service Last.fm, compatible avec de nombreux lecteurs multimédia courants comme Windows Media Player, Winamp, iTunes, Foobar2000 ou encore Mozilla Songbird.

L'application Last.fm est open source et installable sur les systèmes Windows, Mac et Linux, ou encore sur iPod et iPhone.

Pour plus de détails rendez vous sur cette page : http://www.lastfm.fr/download.

Le widget dcScrobbler

Après avoir configuré le plugin, positionnez le widget comme bon vous semble et choisissez le type de flux à afficher.
